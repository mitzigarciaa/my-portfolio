My Projects

Dynamic Blog Website

I built a blog website with multiple pages that lets users easily create, view, edit, and delete their posts. I used JavaScript to handle and store the post data directly in the browser, keeping it simple and accessible. For version control, I used Git, organizing each feature in its own branch to keep things manageable, and hosted everything on GitHub for easy collaboration and project tracking.

Web Storage API

I built a simple web page where users can enter their name, save it using the HTML5 Local Storage API, and have it display again when the page is reloaded. The page includes an input field, buttons to save and clear the name, and a section to show the saved name. I used JavaScript to handle saving the name to local storage, retrieving it when the page refreshes, and clearing it when needed. The design also includes some CSS to make the page look nice and easy to use.

Visit My Web Storage Project
